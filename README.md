# 星TAP 高速图标转换 | StarTAP Icon Converter (Win/Mac)

## 🚀 PNG 转 ICO & ICNS 高清工具

**StarTap Icon Converter** 是一款专为极客和开发者设计的图标转换工具。支持将 PNG 轻松转换为 Windows (`.ico`) 和 macOS (`.icns`) 格式。

---

### 🇨🇳 中文介绍

本工具集成了 Windows 和 macOS 两大平台的图标生成逻辑：
- **Win 版优化**：自动应用 **18% 黄金圆角比**，生成包含 256px 到 16px 的高清多尺寸 ICO 文件，完美适配 Windows 11 现代 UI。
- **Mac 版优化**：自动进行正方形裁剪与居中对齐，生成标准的 Apple ICNS 格式，保持 macOS 原生图标质感。
- **极速体验**：支持拖拽处理，秒级生成，绿色免安装。

### 🇺🇸 English Introduction

**StarTap Icon Converter** is a high-performance tool designed for geeks and developers to create system icons:
- **Windows Optimization**: Automatically applies an **18% rounded corner ratio** and generates high-res ICO files with multiple sizes (from 256px to 16px), perfectly fitting the modern Windows 11 UI.
- **macOS Optimization**: Features automatic square cropping and centering to generate standard Apple ICNS files, maintaining the native macOS look and feel.
- **Fast & Portable**: Drag-and-drop support, instant generation, no installation required.

---

### 🛠️ 核心功能亮点

1.  **智能圆角 (Rounded Corners)**：自动为 Windows 图标添加平滑圆角，告别生硬的直角。
2.  **全尺寸覆盖 (Multi-size)**：ICO 文件内置 256, 128, 64, 48, 32, 16 多种尺寸，确保在任何视图下都清晰。
3.  **双平台支持 (Cross-platform)**：一次转换，同时获得 `.ico` 和 `.icns`。
4.  **LANCZOS 高质量重采样**：采用最高等级的缩放算法，确保小尺寸图标依然锐利。

---

### 🚀 使用指南

#### 【Windows 用户】
1.  下载 `generate_icon_v2.exe` (或运行 Python 源码)。
2.  将你的 PNG 图片命名为 `高速缩图图标.png` 放在程序同级目录，或修改脚本路径。
3.  运行程序，即可在同目录获得 `icon.ico` 和 `icon.icns`。
