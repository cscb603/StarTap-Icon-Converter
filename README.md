## 🚀 快速上手 (Quick Start)

1. **下载**：前往 **[Releases 页面](https://github.com/cscb603/PNG-to-ICNS-Converter/releases)** 下载对应平台的压缩包。
2. **运行**：
   - **Mac**: 解压后直接运行 `png 图标转 icns.app`。
   - **Win**: 直接运行 `图片转ico透明.exe`。
3. **操作**：
   - 将图片拖入软件界面。
   - 勾选相关选项（如 macOS 标准圆角）。
   - 转换后的图标文件会自动生成在原图所在目录。

## 🛠️ 技术细节 (Technical Features)

- **标准 iconset 生成**：自动生成从 16x16 到 512x512@2x 的全尺寸图标集。
- **LANCZOS 高级采样**：缩放过程采用 LANCZOS 算法，确保每一级尺寸都清晰锐利。
- **超采样抗锯齿**：在 4 倍于目标尺寸的空间进行圆角绘制再缩小，边缘质量远超常规转换工具。
